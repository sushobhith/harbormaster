Cheat sheet
  • ⌂ row = one settlement. Chips = the hexes it collects from (colour = resource, number = dice roll).
  • Pips = the dots under a number = how often it rolls. More pips = more cards. That's your whole score.
  • road → = point that settlement's free road here.
  • ⚓ = you start on a port. Counts toward your total, so port players get slightly fewer pips. Still even.
  • ±N at the top = best player minus worst player. 0 or 1 = even game. Above 2 → hit Re-draw.
  • Weakest start plays first. Order is on each card.
Built-in rules
  • 6 and 8 never touch · twin numbers never touch · 2 never touches 12
  • No same-resource clumps · every resource gets a fair pip share
  • Ports fixed to match the real frame · land placed so a 2:1 port avoids its own resource
  • Settlement distance rule respected · 6/8 access evened across players
Physical setup: lay tiles row by row as shown, tokens on top, robber on the desert. Ports are fixed to match the printed frame (photo-verified, both 4-player and 5-6 player) — rotate your frame so the port sequence lines up with the drawn board, then everything matches. Re-draw settlements keeps the board and only re-deals the starting spots and roads. Free and open source — view the code on GitHub.
